## Instalação

Instale o projeto com o Git

```bash
  git clone https://github.com/reddevx/GoshiGo.git

```
Logo após coloque o arquivo .rar no Xampp ou USBWebServer e inicie o apache.
    
## Stack utilizada

**Front-end:** HTML5, CSS3 e Bootstrap

**Back-end:** PHP

**Banco de dados:** Mysql
